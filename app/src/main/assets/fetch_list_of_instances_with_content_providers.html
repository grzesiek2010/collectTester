Uri uri = Uri.parse("content://org.odk.collect.android.provider.odk.instances/instances");
getContentResolver().query(uri, null, null, null, null);

If there are multiple projects, the code above will fetch instances from the first one (the first created).
If you want to use a different project, and you know its ID, you can define it in the URI:
Uri uri = Uri.parse("content://org.odk.collect.android.provider.odk.instances/instances?projectId=3c990f90-72b2-4b73-b157-f7dfc15fc1db");
